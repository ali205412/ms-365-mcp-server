arguments: []

Onboard a new discovery-mode tenant with useful starting memory.

Use `search-tools` to locate core Microsoft 365 operations across mail, calendar, teams, files, users, groups, and security. Use `get-tool-schema` to confirm each candidate's parameters, then call `bookmark-tool` for a small starter set of aliases that are broadly useful. Use `save-recipe` for repeatable workflows such as inbox triage, calendar review, file search, and tenant security review.

Before writing memory, explain each proposed bookmark or recipe and why it helps agents navigate the tenant. Use `execute-tool` only for lightweight verification calls when the user asks for live validation.

Finish with a concise starter catalog grouped by workload and a note about which prompts can reuse the saved recipes.
