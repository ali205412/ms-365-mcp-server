# Virtual Events Workload Guide

Use this guide for webinars, town halls, virtual event sessions, registration, attendance, and organizer workflows.

## Discovery Loop

Start with `search-tools` queries such as "list webinars", "get virtual event registration", "list event sessions", or "read attendance report". Include webinar, town hall, virtual event, or registration terms.

Then use `get-tool-schema` to inspect event ids, session ids, registration ids, organizer context, and paging inputs. Use `execute-tool` after the selected alias matches the event stage.

## Useful Resource Path

Read `mcp://catalog/scope-map.json` to compare event-management and attendance-report permissions before selecting an alias. Endpoint schemas are generated on demand for selected aliases.

## Memory

Bookmark aliases for recurring event operations. Save recipes for event setup checks, registration review, and post-event attendance reporting.
