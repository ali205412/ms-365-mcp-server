# SharePoint Workload Guide

Use this guide for SharePoint sites, lists, list items, document libraries, pages, and site-level content discovery.

## Discovery Loop

Start with `search-tools` queries such as "find site by name", "list site lists", "read list items", or "update list item". Include site, list, item, or document library terms to narrow the catalog.

Then use `get-tool-schema` to identify site ids, list ids, item ids, expansion options, and paging controls. Use `execute-tool` after the selected alias matches the site or list shape.

## Useful Resource Path

Read `mcp://catalog/scope-map.json` before crossing from read-only site discovery into list or item writes. Endpoint schema resources remain generated by alias.

## Memory

Bookmark aliases for the organization's common site and list workflows. Save recipes for recurring list item reports, library searches, and site-content updates.
