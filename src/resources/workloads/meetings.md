# Meetings Workload Guide

Use this guide for online meetings, meeting artifacts, attendance, transcripts, recordings, and meeting-preparation workflows.

## Discovery Loop

Start with `search-tools` queries such as "list online meetings", "get meeting attendance", "read transcript", or "create online meeting". Include meeting, attendance, recording, or transcript terms.

Then use `get-tool-schema` to inspect meeting ids, organizer context, date filters, artifact ids, and paging inputs. Use `execute-tool` when the selected alias matches the meeting artifact needed.

## Useful Resource Path

Read `mcp://catalog/scope-map.json` to understand when meeting metadata, transcripts, recordings, or attendance require different consent. Use generated endpoint schema resources for exact parameters.

## Memory

Bookmark aliases for meeting preparation and post-meeting review. Save recipes for recurring agenda prep, attendance checks, and artifact collection.
